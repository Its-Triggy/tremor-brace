This repository includes the model and print files needed to construct the a tremor brace, as seen here. 
https://www.youtube.com/watch?v=SiWKmoLqlRg

The part "main-support" can be reinforced with 7.5mm aluminum T channel.
https://www.dehobbyspecialist.nl/aluminium-t-profiel-7-5-mm

The copper tee-fitting has a nominal ID of 22mm. 

Parts can be affixed with CA glue. 

Straps can be sewn from 20mm wide fabric and Velcro. Alternatively, large Velcro cable management ties can be modified.
